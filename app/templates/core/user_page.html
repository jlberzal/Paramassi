Bienvenido


Bienvenido a la aplicación de clientes de Paramassi

Actualmente existen en el mercado varios materiales y técnicas que pueden emplearse para mejorar los taludes y laderas. PARAMASSI tiene como filosofía agrupar los materiales en sistemas coherentes, combinando los materiales disponibles para lograr un resultado armónico que resuelva los problemas de manera efectiva.

Con el fin de categorizar y acotar el campo de aplicación de cada uno de estos materiales, PARAMASSI ha definido unos criterios básicos de aplicación para cada tipo de protección, agrupándolos en 6 sistemas diferentes. Con ello se pretende normalizar, clarificar y unificar los criterios de aplicación para cada necesidad, siendo más sencillos para los usuarios y Administraciones.
